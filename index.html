About

Description of Artistic Component: Busyness of the artwork depends on the number of art piece. with larger ammount of details artwork becomes more realistic. Overall, the idea is to show the composition of reality, complexity of reality and its components, the idea of the grows and accumulation of the information, progress, absence of limits. + Reorganization of form, which underlines dynamics, constant change, flexibility and somewhat delicacy of connections.

Structure: Originally the idea was created as a potential prototype for Web Virtual Reality Gallery, which led towards two paths of possible interactions:

1st Path:

  • Full AR/ VR gallery with gesture control
2nd Path:
  • Experience with the gallery using mobile phone as controller (main goal within this path is accessibility from anywhere).

Interaction:

  • 1. Gallery opens by holding a finger on the screen, which opens interface with details (potentially details can contain type of item sorting or choices of specific galleries.
  • 2. When gallery is opened, human is located in the center of the virtual sphere(sketch)
  • 3. Gallery can be explored through phone screen.By targeting an item, the user can fixate it on the screen and access further interface.(sketch)
  • 4. Interaction options with single image are based on usual actions that people can do with images in regular galleries. (screenshot of image, screenshot of the part of the image, slide to next / previous, view details, add personal notes, zoom in out).

Usage:The interface can be used for physical interactive installations, digital installations, where items can be in any other formats

discussion: "This gallery can be understandable and accessible transition between 2dimensional interfaces of the regular gallery and 3dimensional access access to graphic images or other objects.", additional: "The interface can be used for physical interactive installations, digital installations, where items can be in any other formats", further: "improve UI to be more intuitive Finish drawing schemes. Add movement between images. Try to use AR tool
